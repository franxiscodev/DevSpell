# DevSpell - Overview

## ¿Qué es
Sistema de análisis de código Python con IA

## Objetivo de aprendizaje
Dominar Claude Code a nivel profesional

## Stack técnico
- Backend FastAPI + Python 3.11
- Frontend React + TypeScript  
- Base de datos PostgreSQL
- ML Scikit-learn + Claude API
- DevOps Docker + GitHub Actions
- Cloud AWS Free Tier

## Alcance MVP
1. Endpoint que analiza código Python
2. Métricas complejidad, líneas, funciones
3. Sugerencias básicas con IA
4. Dashboard simple
5. Deploy en AWS
